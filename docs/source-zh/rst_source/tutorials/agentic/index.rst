智能体强化学习
===============

本节涵盖使用 RLinf 进行智能体强化学习，包括推理 RL 训练、
多智能体系统和工具集成。

关于端到端示例，请参见 :doc:`智能体示例 <../../examples/agentic/index>` 库，
其中包括推理、在线编码 RL、Search-R1、rStar2 和 WideSeek-R1 工作流。

关键主题：

- **FSDP + SGLang/vLLM/HuggingFace** — 使用 FSDP 后端训练和服务智能体模型。
- **Megatron + SGLang/vLLM** — 使用 Megatron 后端进行大规模智能体训练。
- **推理 RL** — 针对数学和代码推理任务的 GRPO 和 PPO 训练。
- **多智能体 RL** — 原生多智能体训练支持，提供统一的数据接口。
- **工具集成** — 为智能体配备搜索、代码执行等外部工具。
